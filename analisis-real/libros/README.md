# Libros

En esta sección se registran los libros y referencias bibliográficas usadas en el curso **Análisis Real**.

## Aviso

No se alojan libros protegidos por derechos de autor, salvo materiales de dominio público, acceso abierto o con autorización de distribución.

## Referencias principales

- Elon Lages Lima, *Curso de Análisis, Volumen 1*.
- Walter Rudin, *Principios de análisis matemático*.
- Tom M. Apostol, *Análisis matemático*.
