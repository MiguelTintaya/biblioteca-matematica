# Libros

En esta sección se registran los libros y referencias bibliográficas usadas en el curso **Introducción a las Ecuaciones Diferenciales Ordinarias (IEDO)**.

## Aviso

No se alojan libros protegidos por derechos de autor, salvo materiales de dominio público, acceso abierto o con autorización de distribución.

## Referencias principales

- Kreider, Kuller, Ostberg, *Ecuaciones diferenciales*.
