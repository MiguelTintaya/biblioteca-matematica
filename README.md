# Biblioteca Matemática

Este repositorio reúne mis apuntes personales de matemática, ejercicios desarrollados y materiales escritos en LaTeX.

## Contenido

- [Estructuras Algebraicas](estructuras-algebraicas/)
- Análisis Real
- Ecuaciones Diferenciales Ordinarias
- Topología
- Geometría Diferencial
- Recursos bibliográficos

## Objetivo

El objetivo de esta biblioteca es organizar mis avances de estudio, compartir apuntes propios y construir progresivamente material matemático escrito con rigor.

## Aviso importante

Los apuntes publicados aquí son de elaboración propia.  
No se alojan libros protegidos por derechos de autor, salvo materiales de dominio público o con licencia libre.
