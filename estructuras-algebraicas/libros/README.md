# Libros

En esta sección se registran los libros y referencias bibliográficas usadas en el curso **Estructuras Algebraicas (CM2I2A)**.

## Aviso

No se alojan libros protegidos por derechos de autor, salvo materiales de dominio público, acceso abierto o con autorización de distribución.

## Referencias

- Hungerford, T. W. *Algebra*.
- Dummit, D. S. y Foote, R. M. *Abstract Algebra*.
- Herstein, I. N. *Topics in Algebra*.
